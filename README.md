# AELYN FINANCE

A modern multi-user accounting and financial management platform developed by Aelyn Technologies.

![Aelyn Finance Logo](./assets/logo.png)

---

## 📖 Overview

Aelyn Finance is a secure accounting system designed to help businesses manage:

- Revenue
- Expenses
- Profit Distribution
- Investments
- Debt Tracking
- Stakeholder Management
- Financial Reporting

The system supports multiple user roles with synchronized real-time data and detailed activity tracking.

---

# ✨ Core Features

## 💰 Revenue & Expense Management
- Add income records
- Add expense records
- Categorize transactions
- Real-time financial summaries

---

## 📊 Automatic Profit Distribution

The platform automatically distributes profit using the company structure:

- 40% → Company Investment Fund
- 60% → Stakeholder Distribution

Stakeholder distribution:
- Stakeholder A → 30%
- Stakeholder B → 30%

---

## 🧾 Debt Management
- Track company debts
- Automatic 50/50 debt split between stakeholders
- Visual debt indicators:
  - 🔴 Red → Outstanding debt
  - 🟢 Green → Debt cleared

---

## 🔐 Authentication & Security
- Secure login system
- Role-based access
- Separate accounts for:
  - Candidate
  - Chairman
  - Controller
  - Developer

---

## 📝 Activity Logging
Every system action is logged:

- User
- Role
- Action
- Timestamp
- Before/After changes

This ensures transparency between stakeholders.
